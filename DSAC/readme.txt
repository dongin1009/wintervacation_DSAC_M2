밀드 기반 클러스터링 알고리즘
"가까이 있는 샘플들은 같은 군집에 속한다."
인접한 샘플들을 포함시켜 나간다.
--------------------------------------------------
회귀 : 정확한 (수치)를 예측
- 내일의 날씨 예측
- 주가 예측
- 병에 걸릴 확률 예측
- 다음달 가게의 매출 예측
분류 : 주어진 샘플이 (어느 카테고리)에 속하는 지를 예측
- 수신한 메일이 스팸인 지 여부 예측 
- 이번 은행 대출이 부도가 날지 아닐지 여부 예측
- 누가 우수 고객인지 예측

지도학습 : 입력 값(x)과 정답(y, label)을 포함하는 훈련용 데이터를 이용해 학습하고 경사하강법을 이용해 오차를 줄임
비지도학습 : 정답(label)은 없고 입력 데이터만 있는 훈련용 데이터를 이용한 학습을 통해 정답을 찾는 것이 아닌 입력 데이터의 패턴, 특성 등을 발견
-------------------------------------------
정형데이터 : 데이터의 포맷이 정해져 있는 데이터 ex. 엑셀. 대부분의 통계 데이터, 기업의 매출기록 등
비정형데이터 : 일정한 포맷을 가지지 않은 데이터 ex. 블로그 글, 트윗 등 텍스트 문자와, 오디오나 비디오 데이터
반정형데이터 : 완전하게 정의되어 있지는 않지만 조금만 처리를 하면 정형화된 정보를 추출할 수 있는 데이터 ex. 주기적으로 측정되는 센서데이터, 시간대별로 측정한 웹 사용자의 기록

범주형 데이터 : 

데이터 전처리 : 데이터 분석에 앞서 데이터를 정제

결측치 처리 3가지! : 1. 결측치가 포함된 항목을 모두 버리는 방법
	2. 결측치를 적절한 값으로 대체. 0, 평균값, 최소값, 특정 상수, 인접 값으로 추정 등
	3. 분석 단계로 결측치 처리를 넘김(NA로 표기)

틀린 값 처리 : 틀린 값이 포함된 항목을 모두 버리는 방법. 틀린 값을 다른 적절한 값으로 대체. 분석 단계로 틀린 값의 처리를 넘김.
이상치 검출 : 값이 일반적인 범위를 벗어나 특별한 값을 갖는 경우. 데이터 분석 과정의 활동이므로 분석 단계로 넘김.


범주형 변수 코딩 : 
 	one hot encoding : 하나의 특성만 1이 될 수 있고, 다른 특성은 모두 0으로 코딩하는 방법

스케일링 : Min-Max 스케일링 : 주어진 값을 (최소값 = 0, 최대값 = 1) 로 재조정 하는 것
			MinMaxScaler()
	표준 스케일링 : 데이터 분포를 표준 정규분포(평균 0, 표준 편차 1)이 되도록 정규화 하는 방법

클러스터링 : 적절한 군집수, 클러스터링 수
	유사도 : 항목간의 유사한 정도를 수치로 나타낸 것.

	클러스터의 수, K : 적정한 군집의 수(k)를 먼저 찾아야 함
	클러스터링 조건 : 같은 그룹 내의 항목들은 서로 속성이 비슷(유사도가 큼)
